# 安全与隐私

> 最后更新：2026-06-03

## 数据存储

所有数据都在你自己的电脑上：

| 数据类型 | 存储位置 | 说明 |
|---------|---------|------|
| 对话记录 | 本地 SQLite 数据库 | 不上传任何云端 |
| 配置信息 | 本地 SQLite 数据库 | API Key 加密存储，模型配置等 |
| 媒体文件 | `data/channels/` 目录 | 按渠道和会话 ID 隔离存放 |

**没有任何数据离开你的电脑**，除了你主动配置的 AI 模型 API 调用。

## 网络请求说明

HClaw 发起的网络请求只有以下几类：

| 请求类型 | 去向 | 用途 |
|---------|------|------|
| LLM API 调用 | 你配置的模型服务商（DeepSeek、Anthropic 等） | AI 对话推理 |
| 插件安装 | 你指定的 Git 仓库地址 | 安装社区插件 |
| MCP 服务 | 你配置的 MCP 服务地址 | 调用外部工具 |

**除此之外没有其他网络请求。** 你可以用抓包工具验证。

## API Key 安全

- API Key **加密后**存储在本地 SQLite 数据库中，非明文存储
- 仅在向你指定的模型服务商发起请求时解密使用
- HClaw 不会收集、传输你的 API Key 到任何第三方（发送给用户主动配置的服务商是正常行为，不会额外发到别处）。

## 关于源码开放

HClaw **计划在未来开源**，目前正在筹备阶段。

**为什么现在没开源？**
正在筹备完整的安装教程视频和开发者文档，让未来开源时大家能顺利上手、参与贡献。

**当前规划方向：**
- 核心引擎代码在教程体系搭建完成后考虑公开
- 许可证届时可能切换为开源协议
- 开放后会接收社区 PR，共同维护

> ⚠️ 以上规划为当前设想，可能会根据实际情况调整。

在开源之前，安装包仅通过 GitHub / Gitee Releases 页面发布，建议从官方渠道下载，不要使用第三方分发渠道。
