# HClaw `模型方案`配置

## 概述

`模型方案`用于定义 HClaw 调用模型时的参数模板。

## 开始配置

#### 添加模型方案

1. 点击菜单中的`模型`按钮

![click_model_schema.png](../images/click_model_schema.png)

2. 点击`新建方案`按钮

![model_schema_new.png](../images/model_schema_new.png)

3. 配置`模型方案`

![model_schema_detail_config.png](../images/model_schema_detail_config.png)

4. 配置`视觉模型` `听觉模型`

![image_voice.png](../images/image_voice.png)
