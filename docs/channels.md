# 渠道管理

## 概述

HClaw 支持通过多种即时通讯渠道接入，实现跨平台的消息交互。

## 配置路径

进入 **设置 → 渠道管理** 即可管理已连接的渠道。

## 支持的渠道

| 渠道 | 类型 | 说明 |
|-----|------|------|
| 个人微信 | 个人 | 通过微信协议连接 |
| 飞书 | 企业 | 企业级 IM 集成 |
| 钉钉 | 企业 | 阿里系办公集成 |
| 企业微信 | 企业 | 腾讯企业版集成 |

## 连接微信

### 步骤

1. 选择「个人微信」渠道
2. 点击「连接」
3. 使用手机微信扫码登录
4. 等待连接成功

### 注意事项

- 微信账号需已登录网页版
- 部分功能可能受微信政策限制

## 飞书配置

```json
{
  "appId": "cli_xxxxx",
  "appSecret": "xxxxx",
  "botName": "HClaw Bot"
}
```

## 钉钉配置

```json
{
  "appKey": "dingxxxxx",
  "appSecret": "xxxxx",
  "botName": "HClaw Bot"
}
```

## 企业微信配置

```json
{
  "corpId": "wwxxxxx",
  "agentId": "1000001",
  "corpSecret": "xxxxx"
}
```

## 渠道状态

| 状态 | 说明 |
|-----|------|
| 🟢 已连接 | 渠道正常运行 |
| 🔴 已断开 | 连接中断，请检查配置 |
| 🟡 连接中 | 正在建立连接 |

## 消息转发

配置渠道后可设置消息转发规则：
- 指定用户的消息转发到 Agent
- Agent 回复自动发送回原渠道
